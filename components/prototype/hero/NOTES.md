# PROTOTYPE — 히어로 "탈피" 시그니처

**질문:** molt(탈피) 포트폴리오의 히어로에서, 옛 정체성("프론트엔드 개발자")이 웜 허물 가닥으로 *탈피*해 현재의 나("AI를 부려 제품을 끝까지 만드는 엔지니어")가 드러나는 시그니처 모먼트 — 어떤 메커니즘이 가장 좋은가?

**제약:** 따뜻한 라이트 오프화이트 캔버스 + 모노크롬 + 단일 웜 허물 악센트(앰버/코퍼). 애플식 절제·스프링. 스크롤 구동. `prefers-reduced-motion` 폴백. 가벼운 SVG/캔버스(무거운 WebGL 지양). Stripe Sessions의 텍스트-분해 *구조*만 차용, 블루 플루이드 클론 회피.

**변주 (`/prototype/hero?variant=`):**
- **A — 가닥이 글자를 짜고 푼다** (선/SVG): 허물 가닥이 글자를 구성→풀리며 흩어짐→새 정체성 클린 드로잉.
- **B — 허물 한 꺼풀 벗기** (표면/소재): 반투명 허물 시트가 갈라져 벗겨지고, 허물이 가장자리 가닥으로 남음.
- **C — 입자 분해 → 흐름** (입자/플루이드): 옛 글자가 허물 입자로 분해→한 줄기 흐름으로 휩쓸림→새 정체성 해상.

**팔레트:** canvas #f7f4ef · text #1c1917 · sub #78716c · husk-deep #8a4b2a · husk #c2772f · husk-light #e0a96b

## VariantH — 확정 스펙 (2026-06-17 grill-me 재정립)

기존 A~G가 ABCD(비주얼·서사·인터랙션·구도) 전부 불만 → 그릇 자체 재설계. grill-me로 결정한 스펙:

- **그릇:** 추상 풀블리드 효과 폐기 → **구체적 "탈피 사건" 하나**가 서사 짊어짐. 헤드라인은 처음부터 또렷(가독성 1순위 불가침).
- **메커니즘:** 상단 가로 **허물 띠**(웜 앰버) → **자연 스크롤**(핀 없음)하면 비늘 조각이 진짜 중력으로 떨어져 벗겨짐(좌→우 톱니 전선) → 벗겨진 자리에 싱싱한 **하늘색 molt line** 드러남 → 페이지 관통 레일로 이어짐.
- **기술:** 떨어지는 비늘 = **three.js 인스턴스 메시**(풀스크린 셰이더 ❌ = 제네릭 오로라 원인). 불규칙 톱니 가장자리. **통가죽 1장을 조각들이 나눠 샘플링**(per-flake 스티커 ❌).
- **에셋(실빌드):** 허물 albedo 1~2 · 싱싱비늘 albedo 1~2 · **normal · roughness · AO/height · HDRI 환경맵**(이리데센스=환경 반사). 프로토타입은 절차생성 placeholder.
- **색:** 허물 웜 앰버/시에나 → 싱싱비늘 하늘색 주도(초록/앰버 부가). 제네릭 AI 그라데이션 금지.
- **레이아웃:** 2-col 유지(구조는 Stripe Elements와 동일). 마감을 **Stripe식 부드럽게**: 회전 ±7→**±2~4°**, 진한 그림자→**넓게 퍼진 저불투명 소프트**, 풀 그라데이션 카드→**흰 면 + 색은 칩/아이콘/얇은 상단 띠 악센트로만**.
- **폴백:** 데스크탑=풀 3D 낙하 / 모바일·reduced-motion=**정지 1컷**.

**프로토타입 placeholder 한계(실빌드 시 해소):** 비늘 색/질감은 절차생성(텍스처·노멀·HDRI 미적용 → "비싼 재질" 광택 아직 없음). molt line/관통 레일은 CSS 임시. 평가 포인트 = ① 허물이 *벗겨져 떨어지는* 거동/타이밍 ② 좌→우 전선 ③ Stripe식 패널 마감.
